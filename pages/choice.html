Can we quickly set up Cloud9 on your workspace?

Cloud9 needs a few dependencies to be able to work with this SSH workspace.

If you agree, we’ll install just a few things in a ~/.c9 folder:

  • Node.js - to run the Cloud9 basics
  • Tmux - provides smart terminals
  • Pty.js - more terminal dependencies
  • Nak - provides search & file listing

This will not affect any other part of your system, it’s fully self-contained!

How can I revert this later?

To fully remove Cloud9 from your machine, all you need to do is remove the ~/.c9 folder! Nothing will be left over on your system.
